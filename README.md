#CarSearch
 CarSearch is a rest api that will search a local h2 database for the availability of a car.

##Requirements
    Java 11
    Maven 3.6.2
 
##Swagger Doc
http://localhost:8080/swagger-ui.html
  
##Database Console
    http://localhost:8080/h2-console/
    
    jdbc connection: jdbc:h2:mem:cardb
    user:admin 
    password:admin
